KBO Summary 개인정보처리방침

최종 업데이트: 2026년 5월 28일

KBO Summary(이하 "본 앱")는 사용자의 개인정보를 소중히 다루며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 본 앱이 어떤 정보를 수집·이용하며, 어떻게 보호하는지를 안내합니다.

1. 수집하는 정보

본 앱은 회원 가입을 요구하지 않으며, 사용자를 식별할 수 있는 개인정보를 직접 수집하지 않습니다.

항목내용저장 위치
응원 팀 설정 사용자가 선택한 KBO 팀 코드 (예: "LG", "KT") 기기 내부 (AsyncStorage)
온보딩 진행 여부 최초 실행 화면 표시 여부 기기 내부 (AsyncStorage)
광고 식별자 (Advertising ID) 광고 게재용 익명 ID Google AdMob (제3자)
기기 정보 / 네트워크 정보 OS 버전, 광고 노출 등 광고 SDK 가 자동 수집 Google AdMob (제3자)

본 앱 운영자(개발자)는 응원 팀·온보딩 정보를 전송받거나 저장하지 않습니다. 이 정보는 오직 사용자 기기 내부에만 존재하며 앱 삭제 시 함께 제거됩니다.

2. 정보의 이용 목적

  • 응원 팀 설정: 홈 화면에 해당 팀의 경기를 우선 표시
  • 광고 게재: AdMob 을 통한 배너 광고 노출 및 수익화
  • 서비스 안정성 모니터링 (예: 네트워크 오류 추적)

3. 제3자 서비스

본 앱은 다음의 제3자 서비스를 사용합니다. 각 서비스의 개인정보 처리는 해당 사업자 정책을 따릅니다.

서비스제공자용도정책
Google AdMob Google LLC 배너 광고 게재 policies.google.com/privacy
Google Gemini API Google LLC 경기 AI 요약 생성 (서버 측에서 호출, 사용자 식별 정보 전송 X) ai.google.dev/terms
KBO 공식 데이터 (주)한국야구위원회 경기 일정·기록 등 공개 데이터 수집 koreabaseball.com

4. 정보의 보관 및 파기

  • 기기 내부 저장 정보(응원 팀 등): 사용자가 앱을 삭제하면 함께 제거
  • 본 앱 운영자는 별도의 서버에 사용자 식별 정보를 저장하지 않음
  • Google AdMob 의 광고 식별자는 Google 정책에 따라 관리

5. 사용자의 권리

사용자는 언제든 다음 권리를 행사할 수 있습니다.

  • 응원 팀 재설정: 앱 설정 화면 또는 앱 데이터 초기화
  • 맞춤 광고 거부: 안드로이드 기기 설정 → Google → 광고 → "광고 ID 재설정" 또는 "맞춤 광고 사용 안함"
  • 앱 삭제: 모든 로컬 데이터가 즉시 제거됨

6. 아동 보호

본 앱은 만 13세 이상을 대상으로 합니다. 만 13세 미만 아동의 개인정보를 의도적으로 수집하지 않으며, 수집된 사실을 인지한 경우 즉시 삭제합니다.

7. 보안

본 앱은 사용자 식별 정보를 자체적으로 수집·전송하지 않으므로 별도의 인증 절차가 없습니다. 앱이 통신하는 자체 서버는 KBO 공개 데이터만을 다루며, 추후 HTTPS 적용 등 보안 강화를 진행할 예정입니다.

8. 정책 변경

본 방침이 변경될 경우 본 페이지를 통해 사전 고지합니다. 중대한 변경의 경우 앱 내 공지 또는 업데이트 노트를 통해 별도 안내합니다.
